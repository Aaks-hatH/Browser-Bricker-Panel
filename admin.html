Policies Management

Quick Start Guide
  1. Create a policy: Click "Create Policy" above and configure rules (time limits, auto-arm, geofencing, etc.)
  2. Set the scope: Choose if this policy applies to all devices (Global), specific Groups, or individual Devices
  3. Select targets: If using Group or Device scope, you'll see a dropdown to select which groups/devices this applies to
  4. Activate: Policies are active by default. Toggle them on/off using the play/pause icon
